== Key Takeaways

**Start simple.** Most LLM applications only need a chatbot or RAG pattern. Add complexity only when the task demands it.

**Conditional edges are the core primitive.** Every advanced pattern — supervisors, handoffs, loops — is built on conditional edge routing.

**Structured output enables reliable routing.** When the LLM needs to make routing decisions, `with_structured_output` gives you type-safe, parseable results.

**Subgraphs compose.** Complex architectures like team swarms are just simple graphs nested inside each other.
