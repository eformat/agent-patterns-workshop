== Core LangGraph Concepts

**StateGraph** — The foundation of every pattern. A directed graph where nodes are Python functions that read and update shared state.

**Nodes** — Functions that transform state. Each node receives the current state and returns updates.

**Edges** — Connections between nodes. Can be static (always follow the same path) or conditional (choose the next node based on state).

**MessagesState** — A built-in state schema with a `messages` list, perfect for chat-based applications.
